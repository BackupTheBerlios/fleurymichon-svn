\section{BerliOS}

\subsection{Aide au d\'{e}veloppement}

Notre projet est hébergé chez BerliOS, il s'agit d'un fournisseur de services pour développeurs qui nous apporte d'innombrables outils pour travailler sur notre projet. Nous disposons d'une interface assez complète permettant d'organiser l'évolution du  projet, nous pouvons recevoir des demandes d'ajout de fonctionnalités, des demandes de support, émettre des sondages, rapporter les bugs, discuter, publier les releases du projet, des screenshots, de la documentation \ldots La page d'accueil est accessible à l'adresse : 
\begin{center}
\verb+http://developer.berlios.de/projects/fleurymichon/+
\end{center}

\subsection{Repository}

Par ailleurs, BerliOS nous offre également un repository Subversion. Il s'agit d'un gestionnaire de code source, qui nous permet de synchroniser notre travail à la fois rapidement et efficacement. 
\\
TortoiseSVN, un client graphique qui s'intègre à l'explorateur Windows est disponible gratuitement. Les utilitaires svn et svnadmin sont disponibles avec la majorité des distributions Linux (\textit{Voir Fig 4.3}). Le WebSVN accessible à l'URL : 
\begin{center}
\verb+http://svn.berlios.de/wsvn/fleurymichon/+ 
\end{center}
permet de consulter les fichiers sources du projet à partir de n'importe quel navigateur.
